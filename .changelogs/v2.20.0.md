# Release v2.20.0

Released: 2026-06-25

## Highlights

- **Cleaner candles & trend signals.** Live ticks now continue the open candle at the seed/live boundary instead of duplicating it, per-tick volume deltas (not the 24h total) feed the candle cache, non-positive ticker volume can't spike the next candle, and the macro daily-trend slope is read from the daily EMA series rather than a 5-minute snapshot â€” so charts and regime signals read clean and accurate.
- **Accurate Gemini pricing.** Gemini's real-time best bid/ask now track a maintained L2 order book, so they can't ratchet one-way or cross over a session and entries price off the current market.
- **Safer order & cycle accounting.** Consolidation no longer double-sells on a gap-fill, re-places sells when a consolidated order fails (no naked positions), drops `sell_failed` exposure from outstanding aggregates, and the TP-fill backstop matches the `completionPercentage >= 100` rule.
- **More faithful dry-run.** Dry-run entries fill only at/below the limit price, multi-entry cycles weight entry price by per-cycle volume, and immediate/forced saves flush queued funds instead of dropping them.
- **Complete Coinbase fills.** `getOrderFills` paginates via the cursor loop and correctly handles a boolean `size_in_quote`, so fill data is complete and correctly denominated.
- **Hardened config & trade APIs.** The legacy `PUT /api/config` is validated against the allowlist schema, fund-config saves reject cross-market product IDs, the DCA-strategy validator enum matches engine-supported values, UpDown trade routes reject non-numeric cost/returnAmount, and config saves no longer 400 on stale regime keys. Changing dry-run now warns that an engine restart is required.
- **Resilient IPC.** The IPC client wraps frame deserialization in try/catch and guards null frames, so a single malformed frame can't crash the client.
- **Accurate scorecard.** Scorecard reload excludes neutral predictions from `totalPredictions`, matching live counting.
- **Dev UI over Tailscale.** The admin Dev UI loads over a private Tailscale (`*.ts.net`) hostname instead of being blocked as a disallowed host.
- **Dependencies.** Patched `ws` (DoS) and `uuid` (bounds-check) advisories.
