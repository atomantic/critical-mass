# Release v2.1.0 - Dashboard Enhancements & Multi-Asset Support

Released: 2026-01-25

## Overview

This release improves the dashboard UI with better P&L visibility, dynamic asset currency support for non-BTC trading pairs, and various layout refinements.

## New Features

### P&L Metrics
- Added Float P&L display showing unrealized profit/loss in $ and %
- Added Realized P&L display for completed trades
- Added BTC/Asset Market Value card with cost basis comparison

### Multi-Asset Currency Support
- Dashboard now dynamically displays the correct asset symbol based on product ID
- Supports multiple product formats: BTC-USDC (Coinbase), BTCUSD (Gemini), CRO_USD (Crypto.com)
- All labels, values, and tables now use the actual trading asset instead of hardcoded "BTC"

### Fund Assets Display
- Added 3-column layout: Cash + Asset Value = Total
- Shows quote currency (USDC/USD) and asset value separately

## Improvements

### Dashboard Layout
- Compact 2-column grid layout for stats
- Moved Next Trade widget to sidebar for better space utilization
- Moved Last Run info into Next Trade widget
- Added Sync Orders button to pending orders section

### Display Formatting
- Asset quantities now trim trailing zeros (385.65 instead of 385.65000000)
- Avg cost per asset shown with dynamic currency symbol

### Cost Basis Tracking
- Separated cost basis display for pending orders vs reserves
- Shows individual cost basis for each category plus total

## Bug Fixes

- Fixed cost basis calculations to properly separate pending vs reserve holdings
